Compact stat block for the profile and Statistiky grid — big tabular number, label, tinted icon chip.

```jsx
<StatTile value="347" label="Nalezených vozidel" color="var(--cat-tram)" icon={<i data-lucide="tram-front" />} />
<StatTile value="12 480" label="XP celkem" color="var(--xp)" icon={<i data-lucide="zap" />} />
```

Props: `value`, `label`, `icon`, `color`, `center`.
