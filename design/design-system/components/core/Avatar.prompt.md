Player avatar with initials fallback, optional brand ring, and a gold level chip.

```jsx
<Avatar name="Petr Novák" src={photo} size={56} level={12} ring />
<Avatar name="Anna K" size={36} />
```

Props: `src`, `name`, `size` (px), `level` (gold chip), `ring`. For an XP-progress ring around the avatar, use `LevelRing` instead.
